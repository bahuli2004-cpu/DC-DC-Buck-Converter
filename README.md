# ⚡ DC-DC Buck Converter (12V ~ 6V)

A high-efficiency **step-down (buck) converter** for converting a 12V DC voltage into a lower voltage using a PWM control technique.  

## Description
The circuit is a MOSFET-based PWM DC-DC buck converter with an LC filter for an efficient voltage conversion. The circuit configuration is as follows:  
- Input voltage: 12V DC  
- Output voltage: ~5.7V DC  
- Output current: 1 Ω (5.7A)  
- Switching frequency: 25kHz (D = 0.5)  
- LC filter for a low-ripple DC output
